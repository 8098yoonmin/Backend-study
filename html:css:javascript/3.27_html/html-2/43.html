CSS Display: Flexible 박스

flexbox(mdn) 바로가기

  • 유연하게 조정하다는 의미의 새로운 display 속성

  • html5에서 추가 - 모던 브라우저에서 지원

  • flexbox는 부모 요소인 container와 자식요소인 flex item으로 구성됩니다.

  • 컨테이너에 적용하는 속성

    • flex-direction:row ( default ) <-가로배열
    • flex-direction:column <-세로배열
